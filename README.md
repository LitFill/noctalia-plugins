# Noctalia Custom Plugins by LitFill

Custom plugins for [Noctalia Shell](https://github.com/noctalia-dev/noctalia-shell) by [LitFill](https://github.com/LitFill).

## Plugins

| Plugin | Description |
|--------|-------------|
| [niri-auto-center](niri-auto-center/) | Automatically centers the focused window in niri compositor |

## Installation

Add this repository as a custom source in Noctalia settings, or manually symlink plugins:

```bash
ln -s "$PWD/niri-auto-center" ~/.config/noctalia/plugins/niri-auto-center
```

Then enable the plugin in Noctalia settings and restart the shell.
